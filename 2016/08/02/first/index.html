达尔文进化曲

发表于 2016-08-02   |   分类于 情感   |     |   阅读次数

    和先生来到这个城市，走到现在，是如今的我们都没有想到的，这也许就是命运和人为最好的撮合吧！

回想校园里对未来纯纯的憧憬，满满的汗水滋润着梦想的嫩芽。想想，这就是你真正融入社会，才会去细细回味简单美好的那时，当最初衷的情怀消失在流年里，人们总会有一种执着的怀念。

    如今的社会在激烈的进行着达尔文进化曲，尤其是在生物链的越高端，越是严格履行着适者生存，弱肉强食的规律，我们既然选择进军大都市，必然会卷入这无法避免的旋涡。相信很多人和我 们一样，凭着年轻躯壳焕发出的火花，点燃着自己信仰与欲望交织蔓延的草原，就算忍着现实血肉模糊的痛楚，毅然选择爬向最前沿，烽火缭绕的前线，也只有这样才能寻找更残酷的环境更匹配的对手，这样才能激发我们倾尽全力与未来一较高低。结果是赢家的天堂双手欢迎你。

    不得不承认人类总是擅长对美好加冕美轮美奂的皇冠，引发着无穷精神世界的遐想，就像人性深处的点点烛光，照清着善恶美丑融 为一体的我们，有时欲望徘徊不定，就会光影摇曳，模糊了好与坏的界限。让眼前的得失，屏蔽了理性的选择，这样，生活迟早会以某种方式来让我们偿还更多。总之，不要被环境迷惑，我们要时刻保持着自己的准则和这个时代最基本的道德底线平稳衔接，这样才会顺着轨迹滑的更远。

    我和先生的生活是从这里开始，从此开启了人生路途的过山车，这一刻是在低处充满期待，为冲向未来蓄势待发，身心煎熬的是这个时代这个社会这个世界的生存感，但不会影响我们在高点处展望人生的欣喜。
